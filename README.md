# XFit 💪

A full-stack fitness app with AI-powered meal planning.

## 🚀 Features
- Meal Planner
- Dashboard & Analytics
- Chatbot (AI)
- User Profile & Settings

## 🛠 Tech Stack
Frontend: React + Vite + Tailwind  
Backend: FastAPI + SQLite  
AI: Transformers (DistilBERT placeholder)

## 📦 Setup

### Backend
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

### Frontend
cd frontend
npm install
npm run dev
